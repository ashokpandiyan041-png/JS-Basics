# Student Grade Calculator

A simple JavaScript project that calculates a student's grade, pass/fail status, and performance remarks based on marks. This project demonstrates the practical use of core JavaScript concepts such as assignment operators, conditional statements, ternary operators, switch statements, and logical operators.

---

## 📌 Project Overview

The Student Grade Calculator accepts a student's marks, applies bonus marks, determines the grade, checks pass/fail status, and displays performance remarks based on the final score.

This project was built as part of my JavaScript learning journey to strengthen my understanding of programming fundamentals and problem-solving techniques.

---

## 🚀 Features

- Add bonus marks using assignment operators
- Calculate grades using conditional statements
- Determine pass/fail status using a ternary operator
- Generate remarks using a switch statement
- Use logical operators for additional conditions
- Display results in a clean and readable format

---

## 📊 Grade Criteria

| Marks Range | Grade |
|------------|-------|
| 90 - 100 | A |
| 70 - 89 | B |
| 50 - 69 | C |
| Below 50 | F |
